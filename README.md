Flatten PDF
1. First create a folder 
2. Put pdfflatten.sh script in that folder.
3. Put the pdfs that you want to flatten.
4. ./pdfflatten.sh file.pdf or bash pdfflatten.sh
5. Wait some minute, it's done!
